# modern-pictograms-rails 

Modern Pictograms for rails.

## Installation

1) Add this to your `Gemfile`:

```ruby
group :assets do
  gem 'modern-pictograms-rails'
end
```

2) Add this to your `base.css.sass`:

```sass
@import modern-pictograms-rails 
```

## Usage

```haml
%span.modernpics
  d
```
